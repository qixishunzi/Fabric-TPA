# Fabric-TPA

中文 | [English](README.md)

Fabric-TPA 是一个面向 Fabric 服务端的传送模组，适用于生存服与社区服。
它的目标是让普通玩家可以使用 `/tpa` 来替代 `/tp`，同时避免直接授予管理员级别的传送权限。

作者：`Qixishunzi`

## 项目简介

原版 `/tp` 通常只允许管理员使用。
Fabric-TPA 提供了一个适合普通玩家使用的直接传送命令，在保证服务器权限边界的前提下实现常用传送功能。

本模组的核心设计如下：

- 玩家可以直接传送到另一名玩家
- 玩家可以直接传送到指定坐标
- 不需要发送申请，也不需要对方同意
- 管理员可以设置一个全服共享的公共 home
- 所有游戏内传送广播都会显示坐标和维度信息

## 命令

- `/tpa <玩家>`
- `/tpa <x> <y> <z>`
- `/tpa home`
- `/tpa set home`

## 功能说明

- `/tpa <玩家>`：将执行命令的玩家直接传送到目标玩家身边
- `/tpa <x> <y> <z>`：将执行命令的玩家直接传送到指定坐标
- `/tpa home`：将执行命令的玩家传送到全服公共 home
- `/tpa set home`：由管理员在当前位置设置全服公共 home

## 权限说明

- 普通玩家可以使用 `/tpa <玩家>`
- 普通玩家可以使用 `/tpa <x> <y> <z>`
- 普通玩家可以使用 `/tpa home`
- 只有服务器管理员可以使用 `/tpa set home`

## 消息说明

- 游戏内传送广播默认使用中文，并带有颜色格式
- 控制台日志默认使用英文，以提升 Windows 服务端控制台兼容性

## 环境要求

- Minecraft `1.21.11`
- Fabric Loader
- Fabric API
- Java `21` 或更高版本

## 构建

```powershell
.\gradlew.bat build --no-daemon
```

## 备注

- 本模组为纯服务端模组，客户端无需安装
- 公共 home 会保存在世界存档目录中
